# Student Score Analysis

## Description
This project analyzes student performance data to understand score distribution and subject-wise trends using Python.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
1. Data loading and inspection
2. Data cleaning and preprocessing
3. Exploratory Data Analysis (EDA)
4. Visualization of student scores
5. Insight generation

## Key Insights
- Analyzed subject-wise score distribution
- Identified performance trends using visualizations
- Studied relationships between different subject scores

## Note
This is a guided academic project implemented for learning and practice purposes.
